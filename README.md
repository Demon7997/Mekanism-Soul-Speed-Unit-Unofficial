# Mekanism: Soul Speed Unit

This is an **unofficial** module for **Mekanism**, designed to bring Nether-based movement mechanics to your MekaSuit. This project is maintained and updated by Wolf7997.

## Features
* **Soul Speed Unit**: A specialized module for the MekaSuit Boots that grants increased movement speed on Soul Sand and Soil.

### Modules
**Common Modules:**
* **Soul Speed Unit**: Harnesses the power of the Nether to detect the Soul Speed enchantment and apply a speed boost to the MekaSuit when walking on Soul Sand and Soul Soil.

## Additional Features
* **Automatic Detection**: Seamlessly integrates with the Nether enchantment system.
* **MekaSuit Integration**: Easily installable/removable via the standard Mekanism module system.
